# SichtWeisen Website

Unsere Vision bei SichtWeisen ist es, ein praktisches Werkzeug für Coaches, Trainerinnen und Supervisorinnen zu schaffen. Mit Hilfe von Bildern, Worten und kreativen Fragen möchten wir auf spielerische Weise festgefahrene Denkmuster aufbrechen. Unser Ziel ist es, Raum für neue Perspektiven und Handlungsmöglichkeiten zu schaffen.
Als Coach möchten wir den Prozess leicht und sanft lenken, damit die Klientinnen die Lösungen aus sich selbst heraus entdecken und entwickeln können.

[Erfahre mehr auf unserer Website](https://www.sichtweisen.co/)

## Installation

### Installing Ruby & Jekyll

If this is your first time using Jekyll, please follow the [Jekyll docs](https://jekyllrb.com/docs/installation/) and make sure your local environment (including Ruby) is setup correctly.

### Installing

Download or clone the repository.
To run the site locally, navigate to the directory and run:

```
bundle install
```

To start the Jekyll local development server.

```
bundle exec jekyll serve --watch --livereload
```

To build the theme.

```
bundle exec jekyll build
```
